<h3>:pencil:<b> Descripción del Proyecto</b></h3>
<p>
  <b>ORUS</b> es una aplicación de Windows la cual permite llevar un registro de las asistencias del personal de una empresa.<br>
  Permite ingresar Personal en el sistema indicando sus datos personales, como también administrarlo, para que luego puedan regisrtar una <b>entrada</b> o <b>salida</b>.<br>
  También permite la creación de Usuarios a los cuales se les pueden asignar permisos para poder realizar distintas tareas, como:<br>
  &nbsp;&nbsp;&nbsp;- Administracion de Usuarios y Personal<br>
  &nbsp;&nbsp;&nbsp;- Generar reportes para imprimir o en formato .pdf<br>
  &nbsp;&nbsp;&nbsp;- Generar o restaurar copias de seguridad de los datos del sistema<br>
</p>

<h3>:open_file_folder:<b> Acceso e Instalación del Proyecto </b>:wrench:</h3>
<p>
  Para acceder a la instalación del Proyecto se debe descargar el instalador el cual se encuentra en la carpeta "INSTALACIÓN".<br/>
  <b>* Solo se ha probado para Windows 10 x64</b><br/>
  Luego de haberse instalado, se debe ejecutar el acceso directo y brindarle permiso de administrador.<br/>
  Una vez abierto, se deberá seleccionar la opción "Principal".<br/>
  
  <div align="center">
    <img alt="Pantalla instalacion de BD" src="https://github.com/ever077/Orus/assets/38265332/c8528667-551e-444a-8176-36b5324a8a46" width="380" height="280">
  </div>
  
  Si la computadora no tiene instalado SQL Server o SQL Server Express, el programa instalará automáticamente la version "SQLEXPR_x64_ENU"<br/>
  para que el programa pueda funcionar correctamente. Esto se hara de forma automática sin que el usuario requiera presionar botónes.<br/>
  Después de que se haya instalado SQL Server y la Base de Datos se deberá iniciar nuevamente la Aplicación y se requerirá registrar el <b>Usuario Principal</b>,
  el cuál será el <b>Administrador</b> y tendrá acceso a todos los módulos del sistema.

  </div>
</p>
